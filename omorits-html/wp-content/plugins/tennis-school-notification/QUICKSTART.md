# テニススクール開催連絡プラグイン - クイックスタートガイド

## 1. プラグインの有効化

1. WordPress管理画面にログイン
2. 「プラグイン」→「インストール済みプラグイン」を開く
3. 「テニススクール開催連絡」を見つけて「有効化」をクリック

## 2. 初めての開催連絡を作成

### ステップ1: 管理画面を開く
- 左側のメニューから「開催連絡」をクリック

### ステップ2: 情報を入力
1. **日付**: カレンダーアイコンをクリックして日付を選択
   - または「今日」ボタンをクリック

2. **表示色**: 3つから選択
   - 🟢 緑色（通常連絡）: 開催の場合
   - 🟡 黄色（注意連絡）: 注意が必要な場合
   - 🔴 赤色（中止連絡）: 中止の場合

3. **メッセージ**:
   - サンプルメッセージから選択する、または
   - 直接入力する

### ステップ3: 保存
- 画面下部の「保存」ボタンをクリック

## 3. ホームページに表示

### 方法1: 固定ページに追加（推奨）

1. 「固定ページ」→「HOME」（または表示したいページ）を編集
2. 表示したい位置に「ショートコード」ブロックを追加
3. 以下を入力:
   ```
   [tennis_notification]
   ```
4. 「更新」ボタンをクリック

### 方法2: テーマファイルに追加

`wp-content/themes/omori-tennis-club/page-home.php` を編集し、
表示したい位置（例: ヒーローセクションの後）に以下を追加:

```php
<!-- 開催連絡 -->
<div class="school-notification-section">
    <?php echo do_shortcode('[tennis_notification]'); ?>
</div>
```

### おすすめの表示位置

**page-home.php の場合:**

```php
    </section>
    <!-- /.home-hero -->

    <!-- 開催連絡（ヒーローの直後に表示） -->
    <div style="margin: 2rem auto; max-width: 1200px; padding: 0 2rem;">
        <?php echo do_shortcode('[tennis_notification]'); ?>
    </div>

    <!-- 特徴セクション -->
    <section class="features-section">
```

## 4. 動作確認

1. ホームページにアクセス
2. 本日の開催連絡が表示されることを確認
3. 表示されない場合:
   - ブラウザを更新（Ctrl+F5 または Cmd+R）
   - キャッシュをクリア

## 5. 日々の運用

### 毎日の連絡手順

1. 管理画面の「開催連絡」→「新規追加」
2. 今日の日付を選択（「今日」ボタンが便利）
3. 開催状況に応じて色を選択:
   - 開催 → 緑色
   - 条件付き開催 → 黄色
   - 中止 → 赤色
4. サンプルから選択、または直接メッセージを入力
5. プレビューで確認
6. 保存

### 過去の連絡を確認・編集

1. 「開催連絡」→「通知一覧」
2. 一覧から編集・削除が可能

## よくある使い方

### 例1: 通常開催
- **日付**: 今日
- **色**: 緑色
- **メッセージ**: "本日のテニススクールは開催します。"

### 例2: 雨天中止
- **日付**: 今日
- **色**: 赤色
- **メッセージ**: "本日のテニススクールは雨のため中止とさせていただきます。"

### 例3: 一部クラス中止
- **日付**: 今日
- **色**: 黄色
- **メッセージ**: "本日のテニススクールは A・Bクラスは中止とさせていただきます。"

## トラブルシューティング

### Q: 通知が表示されない
**A**: 以下を確認してください
1. プラグインが有効化されているか
2. 本日の日付で通知を作成したか
3. ショートコードが正しく記述されているか
4. ブラウザのキャッシュをクリア

### Q: カレンダーが表示されない
**A**: ページを再読み込みしてください

### Q: プレビューが更新されない
**A**: 日付、色、メッセージのいずれかを変更すると自動更新されます
