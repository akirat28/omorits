# メニュー設定ガイド

## 概要
大森テニスクラブのテーマでは、自動的にメニューが生成される仕組みになっています。WordPressの管理画面でメニューを設定していない場合でも、固定ページを作成すれば自動的にメニューに表示されます。

## メニュー位置

このテーマは2つのメニュー位置をサポートしています：

1. **プライマリーメニュー** - ヘッダーと各ページのサイドバーに表示
2. **フッターメニュー** - フッター部分に表示

## 自動メニュー生成の仕組み

### 重要なページの自動認識

以下のページは自動的に認識され、メニューの上部に優先的に表示されます：

1. **クラブ紹介**
   - 探索順序：
     - URLスラッグ: `club-introduction`
     - URLスラッグ: `about`
     - ページタイトル: `クラブ紹介`

2. **施設案内**
   - ページタイトル: `施設案内`

### メニューの表示順序

1. ホーム（常に最初に表示）
2. クラブ紹介（存在する場合）
3. 施設案内（存在する場合）
4. お問い合わせ（常に最後に表示）

**注意**: サンプページや会員専用ページなど、その他の固定ページは自動的にメニューには表示されません。追加のメニュー項目が必要な場合は、WordPress管理画面でカスタムメニューを作成してください。

## 固定ページの作成方法

### 基本的な手順

1. WordPress管理画面にログイン
2. 左側メニューから「固定ページ」→「新規追加」をクリック
3. タイトルとコンテンツを入力
4. 「公開」ボタンをクリック

### 推奨ページの作成

#### クラブ紹介ページ

```
タイトル: クラブ紹介
パーマリンク: club-introduction または about
内容例:
- クラブの歴史
- 理念・ビジョン
- 代表挨拶
- スタッフ紹介
```

#### 施設案内ページ

```
タイトル: 施設案内
内容例:
- コート数・種類
- クラブハウス設備
- 更衣室・シャワー
- 駐車場情報
```

## カスタムメニューの設定（上級者向け）

自動生成ではなく、メニュー項目を手動で管理したい場合：

### 手順

1. WordPress管理画面で「外観」→「メニュー」を選択
2. 新しいメニューを作成（例：「メインメニュー」）
3. 左側から追加したい項目を選択：
   - 固定ページ
   - 投稿
   - カスタムリンク
   - カテゴリー
4. ドラッグ&ドロップで順序を調整
5. 「メニュー設定」で表示位置を選択：
   - プライマリーメニュー
   - フッターメニュー
6. 「メニューを保存」をクリック

### カスタムメニューの利点

- メニュー項目の順序を自由に変更可能
- 外部リンクを追加可能
- 階層構造（ドロップダウン）を作成可能
- 特定のページをメニューから除外可能

## トラブルシューティング

### メニューが表示されない

1. **テーマが有効化されているか確認**
   - 「外観」→「テーマ」で「大森テニスクラブ」が有効になっているか確認

2. **固定ページが公開されているか確認**
   - 下書き状態のページはメニューに表示されません

3. **ページタイトルを確認**
   - 「クラブ紹介」「施設案内」は完全一致で認識されます

### メニューの順序を変更したい

1. **カスタムメニューを作成**（上記参照）
2. **固定ページのメニュー順序を設定**
   - 固定ページ編集画面で「ページ属性」→「順序」に数値を入力
   - 小さい数値が先に表示されます

### 現在のページがハイライトされない

- 自動的に`current-menu-item`クラスが追加されます
- CSSでスタイリングされているか確認してください

## ページ別メニュー表示

- **HOMEページ**: ヘッダーメニューのみ表示
- **その他のページ**: ヘッダーメニュー + 左サイドバーナビゲーション

## 推奨事項

1. **メニュー項目は5-7個程度に抑える**
   - 多すぎるとナビゲーションが複雑になります

2. **わかりやすいページタイトルを使用**
   - 訪問者が内容を推測できるタイトルにします

3. **重要なページは上部に配置**
   - 自動認識される2つのページ（クラブ紹介、施設案内）を活用

4. **モバイル表示を確認**
   - スマートフォンでの表示も必ず確認してください

## 技術的な詳細

### フォールバック関数

メニューが設定されていない場合、`omori_tennis_fallback_menu()`関数が自動的に呼び出されます。この関数は：

1. 指定された固定ページのみを表示（クラブ紹介、施設案内）
2. サンプページなどの不要なページは自動的に除外
3. 現在のページに`current-menu-item`クラスを追加
4. 適切なCSSクラスを適用（header用/sidebar用）

### ファイル構成

- `functions.php`: `omori_tennis_fallback_menu()`関数の定義
- `header.php`: ヘッダーメニューの表示
- `index.php`, `page.php`, `single.php`, `archive.php`: サイドバーメニューの表示
