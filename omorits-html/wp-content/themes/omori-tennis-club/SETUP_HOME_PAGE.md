# 固定ページ「HOME」の設定手順

## 📋 概要
モダンデザインのトップページを WordPress の固定ページとして展開する手順です。

## ✅ 前提条件
- WordPress がインストール済み
- 「大森テニスクラブ」テーマが有効化済み
- 管理者権限でログイン済み

## 🚀 設定手順

### 1. 固定ページの作成

WordPress 管理画面にログインして以下の手順を実行：

1. **管理画面メニューから「固定ページ」→「新規追加」をクリック**

2. **ページの基本設定**
   - **タイトル**: `HOME` または `ホーム`
   - **パーマリンク**: `home` (自動設定されます)

3. **ページテンプレートの選択** ⭐重要
   - 右側のサイドバー（または「ページ属性」セクション）
   - 「テンプレート」ドロップダウンから「**HOMEページ**」を選択

4. **コンテンツ欄は空のままでOK**
   - page-home.php がすべてのコンテンツを生成します
   - 本文欄に何か入力しても表示されません

5. **「公開」ボタンをクリック**

### 2. フロントページとして設定

1. **「設定」→「表示設定」へ移動**

2. **ホームページの表示を変更**
   - 「**固定ページ**」を選択
   - 「ホームページ」のドロップダウンから、作成した「**HOME**」ページを選択

3. **「変更を保存」をクリック**

### 3. メニューへの追加（オプション）

1. **「外観」→「メニュー」へ移動**

2. **固定ページから「HOME」を選択**
   - 左側の「固定ページ」セクションで「HOME」にチェック
   - 「メニューに追加」をクリック

3. **メニュー構造で位置を調整**
   - ドラッグ&ドロップで最上位に配置することを推奨

4. **「メニューを保存」をクリック**

サイトのトップページ（ホームURL）にアクセスして以下を確認：

## 🔧 トラブルシューティング

### Q: 「HOMEページ」テンプレートが表示されない
**A:** 以下を確認してください：
- テーマが正しく有効化されているか
- `page-home.php` ファイルが正しい場所にあるか
- ファイルの先頭に `Template Name: HOMEページ` が記載されているか
- キャッシュをクリアして再度確認

### Q: デザインが崩れている
**A:** 以下を確認してください：
- `style.css` が正しく読み込まれているか
- ブラウザのキャッシュをクリア
- 他のプラグインとの競合がないか確認

### Q: お知らせが表示されない
**A:** WordPress の「投稿」から記事を1つ以上公開してください

### Q: アニメーションが動作しない
**A:** JavaScriptが有効になっているか確認してください

## 📝 カスタマイズ方法

### 住所・電話番号の変更
以下のファイルを編集：
- `/wp-content/themes/omori-tennis-club/page-home.php` (Line 228-229, 265)
- `/wp-content/themes/omori-tennis-club/footer.php` (Line 5-6)

### 地図の埋め込み
`page-home.php` の Line 285-291 の地図プレースホルダーを Google Maps の iframe に置き換え：

```html
<iframe
    src="https://maps.google.com/maps?q=東京都大田区大森北1-2-3&output=embed"
    width="100%"
    height="100%"
    style="border:0;"
    allowfullscreen=""
    loading="lazy">
</iframe>
```

### カラーテーマの変更
`style.css` で以下の色を変更：
- メインカラー: `#667eea` → お好みの色
- サブカラー: `#764ba2` → お好みの色
