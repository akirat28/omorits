# HEROセクション背景画像の設定ガイド

## 📸 概要
トップページのHEROセクション（メインビジュアル）の背景画像を、WordPress管理画面から簡単に変更できます。
パララックス効果は無効化されており、スクロール時に画像は固定表示されます。

## ✨ 主な機能
- **背景画像のアップロード・変更**
- **オーバーレイの透明度調整**（0〜100%）
- **オーバーレイタイプの選択**（なし、黒、グラデーション）
- **リアルタイムプレビュー**

## 🚀 設定手順

### 1. カスタマイザーを開く

WordPress管理画面から：
1. **「外観」→「カスタマイズ」** をクリック
2. 左側メニューから **「トップページ HEROセクション」** を選択

### 2. 背景画像の設定

#### 画像のアップロード
1. **「HERO背景画像」** の「画像を選択」ボタンをクリック
2. 新しい画像をアップロードするか、メディアライブラリから選択
3. **推奨画像仕様：**
   - サイズ: 1920x1080px 以上
   - 形式: JPG, PNG, WebP
   - ファイルサイズ: 2MB以下を推奨（表示速度のため）

#### 画像の削除
- 「削除」ボタンで画像を削除（デフォルトのグラデーション背景に戻ります）

### 3. オーバーレイの調整

#### オーバーレイタイプ
以下から選択できます：
- **なし**: オーバーレイなし（画像そのまま）
- **黒（シンプル）**: 半透明の黒
- **グラデーション（紫系）**: 紫系のグラデーション ⭐デフォルト
- **グラデーション（青系）**: 青系のグラデーション

#### オーバーレイの透明度
- スライダーで **0%（透明）〜 100%（不透明）** を調整
- **推奨値**: 40〜60%（テキストが読みやすい程度）

### 4. 保存

1. 設定を調整後、**「公開」** ボタンをクリック
2. サイトを確認して完了！

## 🎨 デザインのポイント

### 推奨する画像の選び方

#### ✅ 良い例
- **明るすぎない画像**（テキストが読みやすい）
- **テニス関連の画像**（コート、ボール、プレイシーン）
- **高解像度**（ぼやけない）
- **横長の構図**（デスクトップ表示に最適）

#### ❌ 避けたい例
- 文字が多い画像
- 細かい模様が多い画像
- 縦長の画像
- 低解像度の画像

### オーバーレイ設定の目安

| 画像の明るさ | オーバーレイタイプ | 透明度 |
|------------|--------------|-------|
| 明るい画像 | 黒（シンプル） | 60-80% |
| 中間の明るさ | グラデーション（紫系） | 40-60% |
| 暗い画像 | グラデーション（紫系） | 20-40% |
| 非常に暗い画像 | なし | - |

## 🔧 トラブルシューティング

### Q: 画像が表示されない
**A:** 以下を確認してください：
- 画像が正しくアップロードされているか
- ファイル形式が対応しているか（JPG, PNG, WebP）
- ブラウザのキャッシュをクリア

### Q: 画像がぼやける
**A:** より高解像度の画像（1920x1080px以上）を使用してください

### Q: テキストが読みにくい
**A:** オーバーレイの設定を調整：
- オーバーレイタイプを「黒（シンプル）」に変更
- 透明度を上げる（60%以上）

### Q: モバイルで画像が切れる
**A:** これは正常な動作です。レスポンシブデザインにより、モバイルでは画像の中央部分が表示されます

## 📝 コード上の設定場所

設定値は以下の方法で取得されます：

```php
// functions.php (73-135行目)
// カスタマイザーの設定定義

// page-home.php (13-44行目)
$hero_bg_image = get_theme_mod('hero_background_image', '');
$overlay_opacity = get_theme_mod('hero_overlay_opacity', '50');
$overlay_type = get_theme_mod('hero_overlay_type', 'gradient');
```

## 💡 高度なカスタマイズ

### CSSでの追加調整

`style.css` の359-373行目で以下を調整可能：

```css
.home-hero {
    background-size: cover;      /* 画像のサイズ調整 */
    background-position: center;  /* 画像の位置 */
    background-attachment: fixed; /* スクロール時の挙動 */
}
```

### 背景動画の使用（上級者向け）

動画背景を使用したい場合は、`page-home.php` を直接編集：

```html
<video autoplay muted loop class="hero-video">
    <source src="video.mp4" type="video/mp4">
</video>
```

## 🎯 パフォーマンス最適化

### 画像の最適化
1. **画像圧縮ツール**を使用（TinyPNG, ImageOptim等）
2. **WebP形式**の使用を検討
3. **遅延読み込み**は不要（ファーストビューのため）

### キャッシュの活用
- ブラウザキャッシュを有効化
- CDNの使用を検討（大規模サイトの場合）

## 📞 サポート

問題が解決しない場合の確認事項：
- WordPress バージョン
- テーマバージョン（1.0）
- ブラウザ（Chrome, Safari, Firefox等）
- エラーメッセージの有無
