# 大森テニスクラブ WordPressテーマ

大森テニスクラブの会員制サイト専用テーマです。

## セットアップ手順

### 1. テーマを有効化する

1. WordPress管理画面にログイン
2. 「外観」→「テーマ」に移動
3. 「大森テニスクラブ」テーマを見つけて「有効化」をクリック

### 2. サイト名と説明を設定する

1. 「設定」→「一般」に移動
2. 「サイトのタイトル」に「大森テニスクラブ」と入力
3. 「キャッチフレーズ」に「会員制テニスクラブ」などと入力
4. 「変更を保存」をクリック

### 3. HOMEページを作成する

1. 「固定ページ」→「新規追加」をクリック
2. タイトルに「HOME」と入力
3. 右サイドバーの「ページ属性」→「テンプレート」から「HOMEページ」を選択
4. 「公開」をクリック

### 4. フロントページの表示を設定する

1. 「設定」→「表示設定」に移動
2. 「ホームページの表示」で「固定ページ」を選択
3. 「ホームページ」のドロップダウンから先ほど作成した「HOME」ページを選択
4. 「変更を保存」をクリック

### 5. メニューを設定する（オプション）

1. 「外観」→「メニュー」に移動
2. 新しいメニューを作成
3. 必要なページをメニューに追加
4. 「表示場所」で「プライマリーメニュー」にチェックを入れる
5. 「メニューを保存」をクリック

## テーマの特徴

- レスポンシブデザイン対応
- 会員制テニスクラブに特化したデザイン
- カスタムHOMEページテンプレート
- 清潔感のあるスポーティーなデザイン
- お知らせ表示機能（最新3件を自動表示）
- アクセス情報表示

## カスタマイズ

### 背景画像の変更

デフォルトでは `/wp-content/themes/omori-tennis-club/images/hero.jpg` が背景画像として設定されています。

変更する場合：
1. WordPress管理画面にログイン
2. 「外観」→「カスタマイズ」を選択
3. 「トップページ HEROセクション」をクリック
4. 以下の設定が可能：
   - **HERO背景画像**: トップページのメイン画像をアップロード（推奨: 1920x1080px以上）
   - **オーバーレイの透明度**: 画像の上に重なる色の濃さを調整（0-100）
   - **オーバーレイタイプ**:
     - なし: オーバーレイを表示しない
     - 黒（シンプル）: 黒の半透明オーバーレイ
     - グラデーション（紫系）: 紫系のグラデーション（デフォルト）
     - グラデーション（青系）: 青系のグラデーション

### 色の変更

`style.css`ファイルで以下の色をカスタマイズできます：

- メインカラー: `#1e3c72`, `#2a5298`（青系）
- アクセントカラー: `#4CAF50`（緑系）

### 住所・電話番号の変更

`footer.php`と`page-home.php`の該当箇所を編集してください。

### 地図の埋め込み

`page-home.php`のアクセスセクションにGoogle Maps等の地図を埋め込んでください。

## ファイル構成

- `style.css` - スタイルシート
- `functions.php` - テーマ機能
- `header.php` - ヘッダーテンプレート
- `footer.php` - フッターテンプレート
- `index.php` - デフォルトテンプレート（サイドバー付き）
- `page.php` - 固定ページテンプレート（サイドバー付き）
- `single.php` - 投稿記事テンプレート（サイドバー付き）
- `archive.php` - アーカイブページテンプレート（サイドバー付き）
- `front-page.php` - フロントページテンプレート
- `page-home.php` - HOMEページ専用テンプレート
- `images/hero.jpg` - デフォルトHero背景画像
- `MENU_SETUP.md` - メニュー設定ガイド
