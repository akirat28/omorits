# Images ディレクトリ

このディレクトリは、テーマで使用する画像ファイルを格納するためのフォルダです。

## 📁 ファイル構成

### デフォルト画像の配置（オプション）

HEROセクションにデフォルト画像を設定したい場合：

1. `hero-default.jpg` という名前で画像をこのフォルダに配置
2. 推奨サイズ: 1920x1080px 以上
3. ファイル形式: JPG, PNG, WebP

### その他の画像

テーマで使用する他の画像もこのフォルダに配置できます：
- ロゴ画像
- アイコン
- 背景画像
- その他の装飾画像

## 🎨 画像の最適化

### 推奨ツール
- **TinyPNG** (https://tinypng.com/) - PNG/JPG圧縮
- **Squoosh** (https://squoosh.app/) - WebP変換
- **ImageOptim** (Mac) - 画像最適化

### 推奨設定
- **JPG**: 品質 80-85%
- **PNG**: 圧縮レベル最大
- **WebP**: 品質 80-90%

## 📝 注意事項

1. **著作権**: 使用する画像の著作権を確認してください
2. **ファイルサイズ**: ウェブ表示用に最適化（通常2MB以下）
3. **命名規則**: 英数字とハイフン、アンダースコアのみ使用

## 🔗 画像の参照方法

PHPテンプレート内での画像参照：

```php
<?php echo get_template_directory_uri(); ?>/images/your-image.jpg
```

CSSでの画像参照：

```css
background-image: url('../images/your-image.jpg');
```
